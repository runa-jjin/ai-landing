# 사이트 UI 및 기능 요약

## 🎨 전체 디자인 컨셉
- **배경**: 블랙 (`bg-black`)
- **색상 팔레트**: 블루-틸 그라데이션 (`from-blue-400 to-teal-300`)
- **3D 배경 효과**: 
  - 메인 페이지: 유기적 blob 형태의 3D 오브젝트 (plus-ex.com 스타일)
  - 생성 페이지: 입자 효과 (3,000개)
  - 요금제 페이지: 배경 없음
- **타이포그래피**: 화이트 텍스트, 반응형 폰트 크기

---

## 📄 주요 페이지 구조

### 1. 메인 랜딩 페이지 (`/`)
**목적**: 서비스 소개 및 첫인상 제공

**구성 요소**:
- **Hero 섹션**
  - 메인 헤드라인: "AI가 완성하는 당신의 랜딩페이지 문구"
  - 서브 헤드라인: 브랜드 정보 입력만으로 AI가 문구 생성 설명
  - CTA 버튼 2개:
    - "지금 시작하기" (파란색, `/generate`로 이동)
    - "요금제 보기" (아웃라인, `/pricing`로 이동)
  - 카카오 애드핏 가로형 배너 (728x90) - Hero 상단

**3D 배경**:
- 6개의 유기적 blob 형태 오브젝트
- 보라-핑크-블루 그라데이션 색상
- 자동 떠다니는 애니메이션
- 마우스 따라다니는 효과
- 스크롤 반응형 카메라 움직임
- 셰이더 기반 형태 변형 및 색상 변화

---

### 2. 생성하기 페이지 (`/generate`)
**목적**: 실제 AI 문구 생성 기능 제공

**구성 요소**:
- **헤더 하단 배너 광고**: 카카오 애드핏 가로형 (728x90)
- **메인 컨텐츠 영역** (2단 레이아웃):
  - **좌측**: 
    - 브랜드 정보 입력 폼 (`Form` 컴포넌트)
    - 에러 메시지 표시 영역
    - 카카오 애드핏 세로형 광고 (300x250) - 폼 하단
  - **우측**:
    - 탭 전환 버튼 ("결과 카드" / "미리보기")
    - 결과 표시 영역:
      - 결과 카드: 생성된 문구를 카드 형태로 표시
      - 미리보기: 실제 랜딩페이지처럼 미리보기
- **사용 가이드 섹션**:
  - 4단계 가이드 (입력 → 생성 → 확인 → 커스터마이징)
- **FAQ 섹션**:
  - 6개 자주 묻는 질문 및 답변

**3D 배경**: 입자 효과 (3,000개, 블루-틸 그라데이션)

---

### 3. 요금제 페이지 (`/pricing`)
**목적**: 요금제 비교 및 선택 유도

**구성 요소**:
- **헤더**: "요금제" 제목 및 설명
- **요금제 카드 3개** (그리드 레이아웃):
  1. **스타터** (무료)
     - 월 3회 카피 생성
     - JSON 스키마 기반 결과
     - 브랜드 보이스 요약 제공
  2. **프로** (₩19,000/월)
     - 무제한 카피 생성
     - 팀 공유 링크
     - CSV 내보내기
  3. **에이전시** (₩39,000/월)
     - 다중 브랜드 관리
     - 전담 성공 매니저
     - 커스텀 프롬프트 컨설팅
- **요금제 비교표**: 기능별 상세 비교
- **FAQ 섹션**: 요금제 관련 4개 질문

**3D 배경**: 없음 (깔끔한 읽기 환경)

---

### 4. 결제 안내 페이지 (`/payment-info`)
**목적**: 프로 요금제 결제 프로세스 안내

**구성 요소**:
- **프로 요금제 상세 정보**:
  - 포함된 기능 목록 (6개)
  - 요금 정보 (월 이용료, 결제 주기, 무료 체험, 환불 정책)
- **결제 방법**:
  - 카카오페이 송금 버튼 (QR 코드 링크)
  - 송금 후 이메일 전송 안내
  - 계정 업그레이드 프로세스
- **FAQ 섹션**: 결제 관련 5개 질문

---

### 5. 로그인 페이지 (`/auth/signin`)
**목적**: 사용자 인증 및 서비스 소개

**구성 요소**:
- **서비스 소개 섹션**:
  - 메인 헤드라인 및 설명
  - 3가지 특징 (빠른 생성, 브랜드 맞춤, 완성형 결과)
- **로그인 폼**:
  - Google 로그인 버튼
  - 에러 메시지 표시 (상세한 디버깅 정보 포함)
  - "무료 체험 3회 제공" 안내
- **서비스 특징 섹션**:
  - 4가지 특징 상세 설명 (AI 기반, 완성형 카피, 다양한 업종 지원, 무료 체험)

## 🧩 공통 컴포넌트

### Header (상단 네비게이션)
- **위치**: 고정 헤더 (`fixed top-0`)
- **스타일**: 반투명 배경 (`bg-slate-900/70 backdrop-blur-md`)
- **구성**:
  - 로고 이미지 + 사이트명 (그라데이션 텍스트)
  - 네비게이션 메뉴: 홈, 생성하기, 요금제
  - 로그인 상태에 따른 버튼:
    - 비로그인: "로그인" 버튼
    - 로그인: 이메일 표시 + "로그아웃" 버튼
- **높이**: `py-5` (높이 증가)
- **텍스트 크기**: `text-base` (메뉴 텍스트)

### Footer (하단 푸터)
- **위치**: 모든 페이지 하단
- **구성**:
  - 저작권 표시
  - 개인정보처리방침 링크
  - 이용약관 링크
- **높이**: `pt-8 pb-12` (높이 증가)

### StarryBackground (3D 배경)
- **조건부 렌더링**: 경로별 다른 효과
- **메인 페이지 (`/`)**: 유기적 blob 형태
- **생성 페이지 (`/generate`)**: 입자 효과
- **요금제 페이지 (`/pricing`)**: 배경 없음
- **인터랙션**: 마우스 따라다니기, 스크롤 반응

## 🎨 UI 특징
- **카드 스타일**: `card` 클래스 (어두운 배경, 둥근 모서리, 패딩)
- **그라데이션**: 블루-틸 계열 (`from-blue-400 to-teal-300`)
- **호버 효과**: 버튼 및 링크에 트랜지션 적용
- **로딩 상태**: 스켈레톤 UI (`animate-pulse`)
- **에러 표시**: 빨간색 배경의 경고 박스

---

## 🔐 인증 시스템
- **NextAuth.js**: Google OAuth 로그인
- **세션 관리**: 서버 사이드 세션
- **사용량 제한**: 무료 플랜 월 3회, 프로 플랜 무제한

---

## 📊 데이터 흐름
1. 사용자 로그인 → 세션 생성
2. 브랜드 정보 입력 → OpenAI API 호출
3. 결과 생성 → JSON 스키마로 반환
4. 결과 표시 → 카드 또는 미리보기 형태
5. 복사/다운로드 → 클립보드 또는 파일 다운로드
